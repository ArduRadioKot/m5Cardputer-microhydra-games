# Stack Game

A simple yet addictive stacking game where you need to stack blocks as high as possible!

## Game Description

Stack Game is a challenging arcade-style game where players need to stack blocks on top of each other. The goal is to achieve the highest possible score by creating the tallest tower without letting any blocks fall.

### Game Features

- Increasing difficulty: The game speed gradually increases as you play
- Sound effects: Different sounds for perfect, good, and bad placements
- Score tracking: Keep track of your high score
- Colorful blocks: Each new block has a random color
- Dynamic camera: The view follows your tower as it grows

## How to Play

1. A block moves horizontally across the screen
2. Press ENTER when you want to place the block
3. The new block will be placed on top of the previous one
4. If the blocks overlap, the new block will be cut to the overlapping size
5. If there's no overlap, the game ends
6. Try to stack as many blocks as possible!

### Scoring

- Each successfully placed block adds 1 point to your score
- The game gets progressively faster as your score increases
- Perfect placements (over 90% overlap) play a special sound
- Good placements (over 50% overlap) play a different sound
- Bad placements (less than 50% overlap) play a warning sound

## Technical Requirements

- Hardware: M5 Cardputer
- Firmware: MicroHydra
- Display resolution: 240x135 pixels
- Requires a display and input device
- Sound support for game effects

# Игра Stack Game

Простая, но увлекательная игра, в которой нужно складывать блоки как можно выше!

## Описание игры

Stack Game - это аркадная игра, в которой игроки должны складывать блоки друг на друга. Цель игры - набрать как можно больше очков, построив самую высокую башню, не уронив ни одного блока.

### Особенности игры

- Увеличивающаяся сложность: скорость игры постепенно увеличивается
- Звуковые эффекты: разные звуки для идеального, хорошего и плохого размещения
- Подсчет очков: следите за своим рекордом
- Красочные блоки: каждый новый блок имеет случайный цвет
- Динамическая камера: вид следует за растущей башней

## Как играть

1. Блок движется горизонтально по экрану
2. Нажмите ENTER, когда хотите разместить блок
3. Новый блок будет размещен поверх предыдущего
4. Если блоки перекрываются, новый блок будет обрезан до размера перекрытия
5. Если перекрытия нет, игра заканчивается
6. Постарайтесь сложить как можно больше блоков!

### Подсчет очков

- Каждый успешно размещенный блок добавляет 1 очко
- Игра постепенно ускоряется с увеличением счета
- Идеальное размещение (более 90% перекрытия) воспроизводит специальный звук
- Хорошее размещение (более 50% перекрытия) воспроизводит другой звук
- Плохое размещение (менее 50% перекрытия) воспроизводит предупреждающий звук

## Технические требования

- Оборудование: M5 Cardputer
- Прошивка: MicroHydra
- Разрешение экрана: 240x135 пикселей
- Требуется дисплей и устройство ввода
- Поддержка звука для игровых эффектов
